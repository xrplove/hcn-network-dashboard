# HCN Network Dashboard

**Hernandez Coordination Network - Live Cross-Chain Infrastructure Status**

A comprehensive monitoring dashboard for the Hernandez Coordination Network, providing real-time insights into cross-chain infrastructure health and status.

## Features

- 📈 Live infrastructure metrics
- 🌍 Cross-chain network monitoring
- 🔐 Security and bridge status
- 📊 Performance analytics
- 🎯 Team coordination tools

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection

### Installation

1. Clone the repository:
```bash
git clone https://github.com/xrplove/hcn-network-dashboard.git
cd hcn-network-dashboard
```

2. Serve locally (using Python):
```bash
python -m http.server 8000
```

3. Open in browser:
```
http://localhost:8000
```

## Configuration

Edit `data/config.json` to customize network endpoints and dashboard settings.

## API Integration

The dashboard connects to Guardian-MCP server for real-time metrics.
